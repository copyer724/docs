# 响应式理解及编写

在 vue3 中，使用 ref 或者 reactive 发生变化，都会触发 effect（vue3 内提供的） 函数的执行。

## 理解

响应式的本质：**函数和数据之间的关联**。

- 函数

  - 被 vue 内部监控的函数
  - 列举 vue3 中被监控的函数： render、watch、watchEffect、computed 等等

- 数据
  - 响应式数据（state, props）
  - 数据必须在函数中用到

**当满足上面条件时，在某一时刻，数据发生了变化，函数也就会重新执行**。

:::info 理解

*数据发生变化，页面发生变化*其中的本质就是*数据发生了变化，其 render 函数重新执行了，页面就重新渲染了*
:::
